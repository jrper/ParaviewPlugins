My ParaView Plugins

The Plugins

ShowCVs : Plot control volume data when stored (as per Fluidity's use of the VTK output format) as nodal values on simple unstructured meshes.

VelocitiesOnCVs : Interpolate vector fields flux-style onto the control volume face pseudo-centres. Note that this representation is diagnostic only and doesn't represent the flux values used in the code itself.
